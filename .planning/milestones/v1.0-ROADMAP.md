## Overview

harness 의 UI 층을 Python(prompt_toolkit + Rich)에서 Node + Ink + Zustand + bun + TypeScript 로 전면 재작성. 로컬 사용자 + 외부 원격 2인이 동일한 ui-ink 클라이언트를 사용하고, Python REPL 잔재를 완전 삭제해 Claude Code 급 터미널 UX를 확정한다.

**Core Value 달성:** "ui-ink 가 harness 의 기본이자 유일한 UI. 로컬과 원격이 동일한 경험을 갖고, 그 경험은 Claude Code 수준이다."
