```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  ui-ink:
    name: ui-ink (${{ matrix.os }} / ${{ matrix.runtime }})
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        os: [ubuntu-latest, macos-latest]
        runtime: [bun, node]
    steps:
      - uses: actions/checkout@v4

      - name: Setup Bun
        if: matrix.runtime == 'bun'
        uses: oven-sh/setup-bun@v2
        with:
          bun-version: latest

      - name: Setup Node 22
        if: matrix.runtime == 'node'
        uses: actions/setup-node@v4
        with:
          node-version: '22'

      - name: Install deps (bun)
        if: matrix.runtime == 'bun'
        run: cd ui-ink && bun install --frozen-lockfile

      - name: Install deps (node)
        if: matrix.runtime == 'node'
        run: cd ui-ink && npm ci

      - name: typecheck
        if: matrix.runtime == 'bun'
        run: cd ui-ink && bun run typecheck

      - name: vitest
        if: matrix.runtime == 'bun'
        run: cd ui-ink && bun run test

      - name: guard (금지 패턴 검사)
        if: matrix.runtime == 'bun'
        run: cd ui-ink && bun run guard

      - name: ci:no-escape (alternate screen 가드)
        if: matrix.runtime == 'bun'
        run: cd ui-ink && bun run ci:no-escape

  python:
    name: Python pytest (${{ matrix.os }})
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        os: [ubuntu-latest, macos-latest]
    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-python@v5
        with:
          python-version: '3.14'
          allow-prereleases: true

      - name: Install Python deps
        run: |
          python -m pip install --upgrade pip
          pip install -e ".[dev]" 2>/dev/null || pip install -r requirements.txt 2>/dev/null || true

      - name: pytest
        run: python -m pytest -x --tb=short
```

**보안 (secrets 하드코딩 금지):**
- HARNESS_TOKEN, HARNESS_URL, HARNESS_ROOM 등 실제 토큰은 yml에 포함하지 않음
- 테스트는 모두 Fake WS 서버 기반이므로 실제 서버 연결 불필요
- Python 테스트도 서버 연결 없이 단위 테스트로 동작 (pytest -x)

**Python 3.14 주의:** `allow-prereleases: true` 추가 (3.14는 아직 pre-release 가능)
