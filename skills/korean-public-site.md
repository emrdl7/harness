# 한국 공공기관 웹사이트 관례

## 웹접근성 인증 (WA 인증)
- 한국웹접근성인증평가원(KWCAG) 기준 준수 필수
- WCAG 2.1 AA 이상
- 스크린리더(센스리더, NVDA) 테스트 필요
- 키보드만으로 모든 기능 사용 가능해야 함
- 접근성 자동화 도구: axe, Lighthouse

## 브라우저 지원
- IE 지원 여부는 과업지시서 확인 (최근 대부분 제외)
- Chrome, Edge, Firefox, Safari 최신 2버전

## 공통 UI 패턴
- 건너뛰기 링크(skip navigation) 필수: 페이지 최상단에 `#content`로 이동
- 본문 바로가기, 주메뉴 바로가기
- 인쇄 스타일시트 (`@media print`) 종종 요구
- 사이트맵 페이지 필수인 경우 많음

## 이미지/미디어
- 모든 이미지 alt 텍스트 필수 (공문서 이미지도 포함)
- 동영상에 자막 또는 대본 제공
- 자동재생 금지 (또는 정지 버튼 필수)

## 개인정보/보안
- 개인정보처리방침 페이지 필수
- 주민번호 등 민감정보 입력 시 암호화 필요
- 공공기관은 행정안전부 개인정보보호법 준수

## 납품/검수 시 체크리스트
- 웹접근성 자동 검사 리포트 (Lighthouse 점수 90+ 권장)
- 크로스브라우저 테스트 캡처
- 반응형 화면 캡처 (모바일/태블릿/PC)
- 소스코드 W3C Validator 통과
